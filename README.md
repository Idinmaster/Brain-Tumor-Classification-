# Brain-Tumor-Classification-
Project for digital imaging 

The project divide into 3 part Preprocessing and 2 Deep-learning comparison

Dataset folder contain 3 folder
    Testing/Training : The base image from https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset
    Training_Preprocess : The Training image that had been preprocessed and augmented
    Testing_Preprocess : The Testing image that had been preprocessed only
